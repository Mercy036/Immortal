# Chatbot

A Node.js chatbot application powered by Google Gemini.

## Getting Started

### Prerequisites
- Node.js installed

### Installation

1. Clone the repository
2. Install the necessary dependencies (Node.js equivalent of `requirements.txt`):
   ```bash
   npm install
   ```
3. Set up your environment variables based on the `.env.example` file. Create a file named `.env` in the root of the project and add your `GEMINI_API_KEY`:
   ```env
   PORT=3000
   GEMINI_API_KEY=your_actual_key_here
   ```

### Running the Application

To start the bot, run:
```bash
npm start
```
Or for development (with auto-reload):
```bash
npm run dev
```
